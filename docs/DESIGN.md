# Design Decisions

## Detection

YOLOv8n selected due to:
- Fast inference
- Good accuracy
- Easy deployment

## Tracking

ByteTrack selected because:
- Works well with CCTV footage
- Lightweight
- Simple integration

## Database

SQLite selected because:
- Fast local deployment
- No setup overhead
- Suitable for hackathon environment

## APIs

FastAPI selected because:
- Auto documentation
- High performance
- Production-ready design

## Dashboard

Streamlit selected because:
- Rapid development
- Interactive visualizations
- Easy integration
